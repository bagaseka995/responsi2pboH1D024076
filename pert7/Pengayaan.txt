Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Apa perbedaan antara interface dengan class biasa?
2. Mengapa interface tidak memiliki implementasi method di dalamnya?
3. Apa tujuan penggunaan keyword "implements" pada sebuah class?
4. Apakah sebuah class dapat mengimplementasikan lebih dari satu interface? Jelaskan dan berikan contoh sederhana.
5. Jika sebuah class mengimplementasikan interface, apa yang terjadi jika class tersebut tidak mengimplementasikan semua method dari interface tersebut?

jawaban:
1. perbedaan utamanya adalah class biasa lengkap dari atribut dan method serta dapat dibuat menjadi objek secara langsung. sebaliknya, interface hanya berisi sekumpulan method kosong dan konstanta yang berfungsi sebagai kontrak.
2. karena tujuannya untuk menetapkan standar perilaku subclass, dengan mengosongkan implementasi, interface memberikan kebebasan kepada setiap class yang menggunakannya untuk menentukan logika spesifik yang paling sesuai dengan karakteristik class tersebut.
3. tujuannya adalah untuk membuat class yang identik dengan interface, dan harus mengimplementasi method yang ada pada interface tersebut.
4. bisa. dengan implement bayak iterface, memungkinkan objek mempunyai banyak peran. contoh class hp implement kamera, pemutarMusik. yang berarti hp bisa menjalankan kedua fungsi tersebut.
5. compiler akan menampilkan error
